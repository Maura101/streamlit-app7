Titolo del progetto: Cancerogenicità e relazioni con le caratteristiche molecolari

Questo progetto è un'applicazione web realizzata in Python usando Streamlit per permettere l'esplorazione e la visualizzazione dei dati biochimici interattivamente.

Per avviare il progetto è necessario seguire questi 3 passaggi:

1. Python 3.
2. Librerie necessarie:
numpy
pandas
seaborn
matplotlib
plotly
streamlit

pip install streamlit pandas numpy matplotlib

3. Avviare l'applicazione con:

streamlit run app7.py

Contenuto del progetto:
- app7.py : codice principale della web-app Streamlit
- README.txt : istruzioni per l'uso
- dataset utilizzato
- report python html
